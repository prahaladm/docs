description: "What NativeFlow is and who this documentation is written for."
---

NativeFlow is an AI-native mobile app platform: a drag-and-drop canvas (NativeFlow Studio) and an MCP server that let humans and AI agents build the same apps through the same underlying schema.

This page should orient three distinct readers: architects evaluating whether NativeFlow fits their stack, developers who will build screens and logic in Studio, and AI agents (Claude, Cursor, Copilot) building apps via the MCP server. Link out to System Architecture, Two Ways to Build, and Agent Integration depending on which of those the reader is.
